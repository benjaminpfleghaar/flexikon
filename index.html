Which CSS property can be used to invert the flexbox axes?

  • html
  • css
  • flexbox

Answer

The flex-direction property specifies the direction of the flexible items.

div { display: flex; flex-direction: column; }

Which property in CSS used to set the background color of an element?

  • html
  • css
  • flexbox

Answer

The background-color property sets the background color of an element.

div { background-color: lightblue; }

What HTML tag is used to define an unordered list?

  • html
  • css
  • flexbox

Answer

The <ul> tag defines an unordered list, while <li> represents each list item.

<ul> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ul>
